RoadmapÔÉÅ

In this section a basic feature roadmap is presented. The features in this roadmap are subject to changes. There are no specific delivery dates nor priority table. Some of the unchecked features might already be implemented but not yet documented.

If there is some feature missing that you would like to see added, please check the Contributing section.

  • Supported sensors

    • Baromter

    • GPS

    • IMU (Accelerometer + Gyroscope)

    • Magnetometer

    • Camera (Implemented as a Graph Node)

    • UDP Camera

  • Supported actuators

    • Quadratic Thrust Curve

  • Base vehicles

    • 3DR Iris

    • Typhoon

    • Fixed-wing plane

    • VTOL

  • API backends

    • Mavlink (with direct PX4 integration)

    • Direct ROS 2 interface

    • Python backend for Reinforcement Learning (RL)

  • UI

    • Select from NVIDIA samples worlds

    • Select from Pegasus sample vehicles

    • Add an option to clone and compile PX4-Autopilot directly from the Pegasus Simulator UI
